# Performance

## Web Vitals

Optimize for:

- **LCP** (Large Contentful Paint)
- **CLS** (Cumulative Layout Shift)
- **INP** (Interaction to Next Paint)

## Practices

- Minimize redirects; cache responses; measure server response times.
- Use compression; optimize images and video; use CDNs and web fonts; prefetch/prerender where appropriate.
